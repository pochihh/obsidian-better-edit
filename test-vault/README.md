# Better Edit Test Vault

This folder contains manual regression notes for Better Edit.

Use these files in a clean Obsidian vault with the plugin enabled in Live
Preview. The goal is to exercise interaction-heavy editor behavior that is hard
to validate through pure unit tests alone.

Files:

- `blocks-regression.md`
- `slash-command-regression.md`
- `text-styling-regression.md`
- `image-regression.md`
- `symbol-picker-regression.md`

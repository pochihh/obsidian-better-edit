# Image Arrangement

Image arrangement provides direct image editing controls in Live Preview.

## What users see

- Paste or drop images into the note.
- Resize images with visual handles.
- Align images left, center, or right.
- Crop images from a modal workflow.
- Add or edit captions and alt text.
- Replace an image source without rebuilding the block.
- Arrange multiple images into rows.

## Why it helps

Obsidian's standard image syntax is portable, but editing image layout by hand can be slow. Better Edit adds visual controls for common image tasks while keeping the note content local and inspectable.

## Compact toolbar behavior

When an image becomes narrow, Better Edit collapses controls into a compact menu instead of letting a dense toolbar overflow the image.

## Native-note promise

Better Edit stores image state in standard Markdown image syntax or visible HTML image blocks. HTML image blocks may include Better Edit data attributes and inline styles so the plugin can reopen the editing controls, but the image remains visible without Better Edit.

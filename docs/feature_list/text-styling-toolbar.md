# Text Styling Toolbar

The text styling toolbar makes common inline Markdown formatting available from a selection menu.

## What users see

- Select text in Live Preview.
- A small toolbar appears near the selection.
- Click a formatting action to wrap or unwrap the selection.

## Supported actions

- Bold
- Italic
- Strikethrough
- Highlight
- Inline code
- Inline equation
- Wiki link
- Markdown link

## Why it helps

Inline Markdown syntax is powerful, but repeatedly typing delimiters interrupts writing flow. Better Edit gives the common actions a visual path while still writing normal Markdown.

## Native-note promise

The toolbar writes standard Markdown inline markers such as `**bold**`, `==highlight==`, `` `code` ``, and `[[wiki links]]`.

# Symbol and Emoji Picker

The symbol and emoji picker helps users insert characters that are hard to remember or type directly.

## What users see

Open the picker from any enabled entry point:

- Editor context menu
- Better Edit command
- Configurable shortcut

Then search or browse symbols and insert one at the cursor.

## Included symbol groups

- Greek letters
- Math operators
- Arrows
- Common technical symbols
- Emoji

## Why it helps

Research notes, math notes, and technical writing often need symbols that are not convenient on a standard keyboard. Better Edit keeps those characters close to the editor without requiring snippets or external character maps.

## Native-note promise

The picker inserts ordinary Unicode text. Notes remain portable and readable without Better Edit.
